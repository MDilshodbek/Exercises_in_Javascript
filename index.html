1. Problem: Print Numbers from 1 to n Write a program that takes an input n from the user and prints all the numbers from 1 to n on separate lines.
2. Problem: Find Even or Odd Write a program that takes an input num from the user and checks if it is even or odd. Print "Even" if the number is divisible by 2, otherwise print "Odd".
3. Problem: Sum of Even Numbers Write a program that takes an input n from the user and calculates the sum of all even numbers from 1 to n.
4. Problem: Positive, Negative, or Zero Write a program that takes an input num from the user and prints "Positive" if the number is greater than 0, "Negative" if the number is less than 0, and "Zero" if the number is 0.
5. Problem: Multiplication Table Write a program that takes an input n from the user and prints the multiplication table of n from 1 to 10.
6. Problem: Sum of Digits Write a program that takes an input num from the user and calculates the sum of its digits.
7. Problem: Find Prime Numbers Write a program that takes an input n from the user and prints all prime numbers from 1 to n.
8. Problem: Factorial Calculation Write a program that takes an input n from the user and calculates its factorial.
9. Problem: Check Leap Year Write a program that takes an input year from the user and checks if it is a leap year or not. A leap year is divisible by 4, and either not divisible by 100 or divisible by 400.
